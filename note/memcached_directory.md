# Memcached源码分析目录

徐顺 2013-08-10

## 大纲

`Memcached`是一个高性能的分布式内存对象缓存系统，用于动态 Web 应用以减轻数据库负载。  
此系列文章通过阅读源码对`Memcached1.4.15`的核心部分进行了解析，希望和对Memcached感兴趣的朋友们一起学习，共同进步。在阅读文章中，如有建议敬请反馈，谢谢。
